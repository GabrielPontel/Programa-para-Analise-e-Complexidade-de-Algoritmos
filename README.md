# Análise e Implementação de Algoritmos

Este projeto reúne implementações de diversos algoritmos clássicos da Ciência da Computação, desenvolvidos com o objetivo de estudar seu funcionamento, desempenho e complexidade computacional. Além da implementação prática, o projeto busca comparar diferentes abordagens para resolução de problemas e analisar sua eficiência por meio da contagem de operações realizadas.

## Objetivos

- Implementar algoritmos fundamentais da computação.
- Comparar diferentes estratégias de resolução de problemas.
- Analisar complexidade temporal e espacial.
- Avaliar desempenho através da contagem de operações.
- Estudar estruturas de dados amplamente utilizadas.
- Compreender o impacto de diferentes abordagens sobre eficiência e escalabilidade.

## Conteúdo do Projeto

### Algoritmos de Busca

- Busca Linear
- Busca Binária

Permitem localizar elementos em coleções de dados e comparar a eficiência entre abordagens sequenciais e divididas.

### Algoritmos de Ordenação

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

Implementações utilizadas para comparar desempenho, número de operações e complexidade assintótica.

### Sequência de Fibonacci

Diferentes abordagens para cálculo da sequência:

- Iterativa
- Recursiva
- Memoização

Possibilitando observar o impacto da otimização por armazenamento de resultados.

### Fatorial

Implementações para cálculo do fatorial de um número utilizando diferentes abordagens:

- Iterativa
- Recursiva

Permitindo analisar diferenças entre métodos recursivos e iterativos para um mesmo problema.

### Números Primos

Implementações para identificação e geração de números primos:

- Verificação simples de primalidade
- Crivo de Eratóstenes
- Crivo Segmentado

### Estruturas de Dados

- Lista Encadeada (Linked List)
- Pilha (Stack)
- Fila (Queue)

Estruturas utilizadas como base para diversos algoritmos e aplicações.
